# webext-animation-lib — Animations for Extension UIs
> **Built by [Zovo](https://zovo.one)** | `npm i webext-animation-lib`

Promise-based animations: fade, slide, pop, shake, pulse, and highlight.

```typescript
import { Animate } from 'webext-animation-lib';
await Animate.fadeIn(element);
await Animate.shake(errorField);
await Animate.pop(button);
```
